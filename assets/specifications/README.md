Carpeta para capturas de especificaciones del menu.

del catalogo y el cliente ya recibio el menu PDF antes.

- toppings.png, toppings.jpg, toppings.jpeg o toppings.webp
- adiciones.png, adiciones.jpg, adiciones.jpeg o adiciones.webp
- sabores.png, sabores.jpg, sabores.jpeg o sabores.webp
- salsas.png, salsas.jpg, salsas.jpeg o salsas.webp
- frutas.png, frutas.jpg, frutas.jpeg o frutas.webp
- productos.png, productos.jpg, productos.jpeg o productos.webp
- precios.png, precios.jpg, precios.jpeg o precios.webp
